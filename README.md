# Materia-oscura-Curvas-de-rotaci-n-de-Galaxias-y-Estad-stica-Bayesiana
Material educativo sobre materia oscura, curvas de rotación y aplicaciones de estadística bayesiana a estos tópicos.

En este notebook se aborda el concepto de materia oscura, la evidencia observacional relacionado a ella, se profundiza específicamente en el tópico de las curvas de rotación de las galaxias, la aplicación que tienen los algoritmos MCMC en esta área y se desarrollan dos ejemplos de implementación de EMCEE utilizando conjuntos de datos reales obtenidos del proyecto SPARC.

El propósito en general es que sirva como una herramienta para aquellos estudiantes de de nivel medio superior o primeros semestre de licenciatura en física o matemáticas que quisieran conocer un poco más sobre estos tópicos, el estado actual del problema, las bases de datos que existen y cómo se puede trabajar con esos datos utilizando materiales que les sean accesibles si conocen un poco de programación en Python, así como las alternativas que existen en R y Julia.

Los datos con los que se trabaja en el notebook son del proyecto SPARC, en la bibliografía se puede encontrar el repositorio del que se obtuvo así como el documento en JOSE (Journal of Open Source Education) de sus creadores.
